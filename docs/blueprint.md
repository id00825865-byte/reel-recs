# **App Name**: ReelRecs

## Core Features:

- Recommendation Input: A prominent input field where users can describe their movie preferences (genres, actors, similar movies, mood, etc.) in natural language.
- AI Movie Recommendation Engine: An AI-powered tool that analyzes user text input and leverages its reasoning capabilities to select and recommend a minimum of 3 relevant movies.
- Movie Details Display: For each recommended movie, display essential information including the movie poster, title, synopsis, director, and key cast members.
- Dynamic Search Results: Display movie recommendations in a clear, card-based format that dynamically updates after each user query.

## Style Guidelines:

- Primary color: A warm, deep bronze (#93591F), chosen to evoke the classic, sophisticated feel of a cinema while adding warmth.
- Background color: A very dark, almost black hue with a hint of warmth (#231F1A), creating a cozy, immersive cinematic environment ideal for movie watching.
- Accent color: A vivid, clear red (#ED5E5E) that provides strong contrast for interactive elements and calls to action, standing out against the dark background.
- Headlines and movie titles: 'Playfair' (modern serif) for an elegant, high-end, and classic cinematic aesthetic. Body text and descriptions: 'PT Sans' (humanist sans-serif) for excellent readability and a modern, approachable feel.
- Use sleek, minimalist icons, focusing on elements relevant to movies such as play buttons, search glass, and film reels.
- Implement a clean, grid-based layout for displaying movie recommendations, ensuring each movie card is easily digestible and visually appealing. The main search input will be central and prominent.
- Subtle fade-in and slide-up animations for movie recommendation cards as they appear on screen, enhancing the feeling of discovery and responsiveness.
